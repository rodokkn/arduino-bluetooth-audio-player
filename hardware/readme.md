# Hardware

This folder contains all hardware-related files for the Bluetooth Audio Player project.

The hardware design focuses on a compact, portable, and rechargeable audio system.

---

## Main Components

- ESP32 microcontroller
- 7.5" TFT display
- PAM8302 audio amplifier
- Passive speaker
- Rechargeable Li-ion battery
- Charging module
- Voltage regulator
- Control button and potentiometer

---

## Folder Structure

- `cad/` – SolidWorks enclosure design files  
- `stl/` – 3D printable enclosure files  
- `schematics/` – Wiring diagrams  
- `photos/` – Project photos

---

## Notes

All components were selected and wired for low power consumption and portable use.
