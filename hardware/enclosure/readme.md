# Enclosure

This enclosure was designed to hold all components of the Bluetooth Audio Player in a compact and simple form.

- Designed in **SolidWorks**
- Printed using **PLA** material
- Suitable for **FDM 3D printing**
- Fits ESP32, amplifier, speaker, display, and control components

---

## Files

- `enclosure.SLDPRT` – SolidWorks design file  
- `enclosure.stl` – Ready-to-print STL file
